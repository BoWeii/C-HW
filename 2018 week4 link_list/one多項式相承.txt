33. 
Link List多項式 

使用 Link List，輸入兩個多項式，輸出相乘的結果。例如: 

2x^4 + 3x^3 + x -1 
x^5 - x^3 + 4x^2 - 3x +2 

結果： 

2x^9 + 3x^8 - 2x^7 + 6x^6 + 5x^5 - 6x^4 + 11x^3 - 7x^2 + 5x - 2 

----------- 
輸入說明 

輸入兩筆資料，分別代表兩個多項數。 
每一筆輸入 n 個整數，第一個代表 n-1次方的係數，第 n 個代表 0 次方係數。 

---------- 
輸出說明 

兩個多項式相乘後從最高次方到0次方的係數，相乘結果為0的係數也需印出。 
---------------- 
Sample Input 

2 3 0 1 -1 
1 0 -1 4 -3 2 

------------ 
Sample Output 
2 3 -2 6 5 -6 11 -7 5 -2
